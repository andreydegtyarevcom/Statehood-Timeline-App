# Statehood Timeline App

An interactive web application that visualizes the historical origins of all modern countries.

The app presents a chronological timeline of state formation, starting from ancient civilizations to modern nation-states. Countries are sorted by their earliest known historical roots, allowing users to explore how states emerged over time.
